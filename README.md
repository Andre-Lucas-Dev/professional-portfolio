# Responsive Personal Portfolio Website

Este é um **site de portfólio pessoal responsivo**, desenvolvido com **HTML, CSS e JavaScript**, que apresenta um design moderno, animações suaves e uma interface agradável para todos os dispositivos.

## Funcionalidades

- **Animações ao rolar a página:** Elementos aparecem com animações suaves à medida que o usuário faz scroll.  
- **Scroll suave entre seções:** Navegação fluida e agradável entre as seções do site.  
- **Mobile First:** Desenvolvimento inicial focado em dispositivos móveis, garantindo uma ótima experiência em telas menores.  
- **Compatível com todos os dispositivos:** Layout adaptável para smartphones, tablets e desktops.  
- **Interface bonita e agradável:** Design moderno e intuitivo, com atenção à experiência do usuário.

## Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
